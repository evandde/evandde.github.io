리눅스 시스템 관련 명령어 모음

마지막 업데이트 Sep 4, 2021 1 분 길이 Linux

리눅스 운영체제에서 하드웨어 및 시스템의 정보나 현재 상태를 살펴보는 데에 사용되는 명령어를 정리하였습니다.

OS 확인

lsb_release -a

메모리 및 swap 영역 상태 확인

free -h

디스크 용량 확인

df -h

파티션 정보 확인

lsblk

CPU 모델명 확인

cat /proc/cpuinfo | grep CPU | head -1

CPU 코어 수 확인

cat /proc/cpuinfo | grep CPU | wc -l

서버 상태 확인

w
