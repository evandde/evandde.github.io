작업표시줄, 시작메뉴에서 최근 항목 뜨지 않게 하는 법

마지막 업데이트 Sep 4, 2021 1 분 길이 Windows

작업표시줄에서 아이콘을 우클릭할 때 뜨는 팝업이나, 시작메뉴의 점프목록 등에서 나타나는 최근 항목에 대한 기록을 없애는 방법에 대해 알아봅니다.

해결방법

  1. [WIN]+[I]를 눌러, Windows 설정 창을 엽니다.

  2. 개인 설정에 들어갑니다.

  3. 왼쪽 탭 중 시작 탭에 들어갑니다.

  4. 시작 메뉴의 점프 목록, 작업 표시줄 또는 파일 탐색기 즐겨찾기에서 최근에 연 항목 표시의 스위치를 클릭하여 끔으로 바꾸어줍니다.

끝입니다! 설정사항은 따로 확인버튼 없이 자동으로 저장되니 그냥 창을 끄셔도 됩니다!
