Geant4 설치 예제 링크 및 설명

이 글에서는 각 설치 예제마다 세부 옵션과 설치 목표를 확인할 수 있습니다.

확인하시고 본인의 상황에 적합한 예제를 참고하세요.

Windows

Windows 10에서 10.7 patch-02 설치하기(초보자용)

  • Windows 10 환경에서 Windows용 Geant4 설치
  • Microsoft Visual C++ (MSVC) 빌드 도구 사용
  • multi-threading 기능 활성화
  • 외부 라이브러리 Qt5 (그래픽/UI 라이브러리) 연동
  • CLHep 내장 라이브러리 사용
  • Cross-section Data 설치 중 자동 다운로드

Ubuntu

10.6 patch-02 설치가이드

  • cmake 이용 (ccmake는 사용하지 않음)
  • multi-threading 기능 활성화
  • 외부 라이브러리 Qt5 (그래픽/UI 라이브러리) 연동
  • 외부 라이브러리 OpenGL (그래픽) 연동
  • CLHep 내장 라이브러리 사용
  • Cross-section Data 설치 중 자동 다운로드

10.7 patch-01 설치가이드

  • cmake/ccmake 이용
  • multi-threading 기능 활성화
  • 외부 라이브러리 Qt5 (그래픽/UI 라이브러리) 연동
  • 외부 라이브러리 OpenGL (그래픽) 연동
  • 외부 라이브러리 CLHep 연동
  • Cross-section Data 직접 다운받아 준비

WSL

Windows 10에서 WSL 활용하여 10.7 patch-01 설치하기(초보자용)

  • Windows 10 환경에서 WSL1 설치부터 진행
  • cmake 이용 (ccmake는 사용하지 않음)
  • multi-threading 기능 활성화
  • 외부 라이브러리 Qt5 (그래픽/UI 라이브러리) 연동
  • 외부 라이브러리 OpenGL (그래픽) 연동
  • CLHep 내장 라이브러리 사용
  • Cross-section Data 설치 중 자동 다운로드
